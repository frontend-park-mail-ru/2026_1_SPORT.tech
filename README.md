## О проекте

Клиентская часть платформы спортивного контента **SPORT.tech**.

**Стек:** Vanilla TypeScript + Handlebars + Webpack 5 + Babel (без UI-фреймворка).
Кастомный клиентский роутер, атомарный дизайн компонентов, общение с бэкендом
через `/api`.

## Быстрый старт

```bash
npm install
npm run dev          # дев-сервер на http://localhost:8080
```

## Разработка

```bash
npm run build        # продакшн-сборка → dist/
npm run type-check   # проверка типов (без эмита)
npm run lint         # ESLint
npm run lint:fix     # ESLint с авто-фиксом
```
